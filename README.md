# 关于这个项目
就如简介所说，这是个用来练手的socket项目

## 构成
主体：MySocket  
更新器：MySocket-updater  
<https://github.com/kao-chinaklp/MySocket-Updater>

## 关于本体
本体使用了curl、json、openssl和mysql的库，更新器使用了curl和json的库，但库本体不保留在项目中，编译/运行的时候记得自行添加。
